# Markdown

- ist eine einfache, vielgenutzte Auszeichnungssprache (engl. markup
  language)
- kann in HTML, PDF un vielen andere übersetzt werden (e.g. mit dem
  Werkzeug pandoc)

## Überschriften

```
# Überschriften 1
## Überschriften 2
### Überschriften 3
#### Überschriften 4
##### Überschriften 5
###### Überschriften 6
```

## Betonungen

```
- *kursiv* / _kursiv_
- **fett** / __fett__
- ***fett-kusib***
- ~~durchgestrichen~~
```
